# Solution

## Task 1

Given:

Sample space:
Ω = {ω₁, ω₂, ω₃, ω₄, ω₅}

Events:
A = {ω₁, ω₃, ω₅}  
B = {ω₂, ω₃, ω₄}

---

### 1. Union of events (A ∪ B)

The union contains all elementary outcomes that belong to A or B (or both).

A ∪ B = {ω₁, ω₂, ω₃, ω₄, ω₅}

Explanation:  
A contains ω₁, ω₃, ω₅  
B contains ω₂, ω₃, ω₄  
Combining all distinct elements gives the entire sample space.

---

### 2. Intersection of events (A ∩ B)

The intersection contains all elementary outcomes that belong to both A and B.

A ∩ B = {ω₃}

Explanation:  
The only common element between A and B is ω₃.

---

### 3. Difference of events (B ∖ A)

This contains elements that belong to B but not to A.

B ∖ A = {ω₂, ω₄}

Explanation:  
B = {ω₂, ω₃, ω₄}  
Removing ω₃ (since it belongs to A) leaves {ω₂, ω₄}.

---

### 4. Difference of events (A ∖ B)

This contains elements that belong to A but not to B.

A ∖ B = {ω₁, ω₅}

Explanation:  
A = {ω₁, ω₃, ω₅}  
Removing ω₃ (since it belongs to B) leaves {ω₁, ω₅}.
