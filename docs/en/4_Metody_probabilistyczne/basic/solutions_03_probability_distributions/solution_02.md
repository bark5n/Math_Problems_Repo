## Task 2 — Hypergeometric Model (Sampling from a Batch)

### Random Experiment

We select 4 components without replacement from 20.

---

### Random Variable

X = number of defective components

---

### Possible Values

X ∈ {0,1,2,3,4}

---

### Probability Distribution

P(X = k) = [C(5,k) · C(15,4−k)] / C(20,4)

---

### Success Definition

Success = selecting a defective component

---

### Key Idea

No replacement → hypergeometric model
