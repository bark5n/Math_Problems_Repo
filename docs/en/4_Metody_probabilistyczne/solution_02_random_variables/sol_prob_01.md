## Task 1 – Random Variable: Student Grades

**Problem:**

A test was conducted in a student group.  
Let **X** denote the grade (on the grading scale 2–5) of a randomly selected student.

We want to determine whether **X** is a random variable and find the probability of obtaining each grade.

---

### Is X a random variable?

Yes. **X is a random variable** because it assigns a numerical value (the grade) to each possible outcome of the experiment (randomly selecting a student).

Possible values of X:

X ∈ {2, 3, 4, 5}

---

### Given grades

5, 4, 3, 3, 4, 5, 3, 3, 4, 2

Total number of students = 10

The probability of each grade is calculated using relative frequency:

P(X = x) = (number of occurrences of grade x) / 10

---

### Probability distribution

Grade 2

Number of occurrences = 1

P(X = 2) = 1/10

---

Grade 3

Number of occurrences = 4

P(X = 3) = 4/10

---

Grade 4

Number of occurrences = 3

P(X = 4) = 3/10

---

Grade 5

Number of occurrences = 2

P(X = 5) = 2/10
