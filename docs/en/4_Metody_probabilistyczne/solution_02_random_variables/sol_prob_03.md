## Task 3 – Probability Function of Costs

**Problem:**

The monthly cost **u** of running an example laboratory depends on the number **x** of employees hired.  
The cost function is:

u = 15000x + 10000√x

The number of employees is treated as a random variable **X** with the following distribution:

| xi | 2 | 3 | 4 | 5 |
|----|---|---|---|---|
| pi | 0.10 | 0.25 | 0.40 | 0.25 |

We want to determine the **probability function of the cost**, which is the random variable **U**.

---

### Step 1 – Calculate the cost values

We compute **u = 15000x + 10000√x** for each value of **x**.

For x = 2

u = 15000(2) + 10000√2  
u ≈ 30000 + 14142  
u ≈ **44142**

---

For x = 3

u = 15000(3) + 10000√3  
u ≈ 45000 + 17320  
u ≈ **62320**

---

For x = 4

u = 15000(4) + 10000√4  
u = 60000 + 20000  
u = **80000**

---

For x = 5

u = 15000(5) + 10000√5  
u ≈ 75000 + 22361  
u ≈ **97361**

---

### Step 2 – Probability distribution of U

Since **U depends directly on X**, the probabilities remain the same.

| U (cost) | 44142 | 62320 | 80000 | 97361 |
|----------|-------|-------|-------|-------|
| P(U)     | 0.10  | 0.25  | 0.40  | 0.25  |

---

### Final Result

The **probability function of the cost U** is:

P(U = 44142) = 0.10  
P(U = 62320) = 0.25  
P(U = 80000) = 0.40  
P(U = 97361) = 0.25
