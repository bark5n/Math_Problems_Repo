## Task 9 – Density of a Two-Dimensional Random Variable

**Problem:**

Find constant **c** such that the function

f(x,y) = cxy  for 0 ≤ x ≤ 2 and 0 ≤ y ≤ 1  
f(x,y) = 0    otherwise

is the density of the random variable (X,Y).

---

### Step 1 – Normalization condition

For a density function:

∬ f(x,y) dx dy = 1

Therefore:

∫₀² ∫₀¹ cxy dy dx = 1

---

### Step 2 – Integrate with respect to y

∫₀¹ cxy dy

= cx ∫₀¹ y dy

= cx [y²/2]₀¹

= cx/2

---

### Step 3 – Integrate with respect to x

∫₀² cx/2 dx

= c/2 ∫₀² x dx

= c/2 [x²/2]₀²

= c/2 · 4/2

= c

---

### Step 4 – Solve

c = **1**

---

### Final Result

c = **1**

f(x,y) = xy  for 0 ≤ x ≤ 2 and 0 ≤ y ≤ 1
